Installation

Currently the dm Server Tools support Eclipse 3.3 and 3.4 with the corresponding version of WTP. Downloading and unzipping the Eclipse IDE for Java EE Developers is the easiest way to start.

Execute the following steps to install the dm Server Tools into your Eclipse.

  1. Install Spring IDE 2.0.5 or later from http://springide.org/updatesite/ using the Eclipse Update Manager.

    Note
    Don't try to install the "Spring IDE Dependencies (only for Eclipse 3.2.x)" from the "Dependency" category on Eclipse 3.3 or 3.4. This feature is intended only for Eclipse 3.2 and is to keep Spring IDE backward-compatible. You will not be able to continue with the installation if you select this feature on Eclipse 3.3 or 3.4.

  2. Install the SpringSource dm Server Tools from http://static.springsource.com/projects/sts-dm-server/update/ using the Eclipse Update Manager.
