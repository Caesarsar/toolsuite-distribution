New Bean Wizard

The New Bean wizard helps you create a new bean configuration in a spring configuration file.

Bean definition page

Option Description Default
Bean definition file Type the location of the bean configuration file that the new bean definition belongs to. Click on Browse... to browse for existing bean configuration file.
From new wizard
<blank>
From Spring Explorer
the selected bean configuration file
From Bean Configuration Editor
the bean configuration file that the editor is opened with
Id Type an ID for the new bean definition. <blank>
Name Type a name for the new bean definition. <blank>
Class Type the fully qualified name of the Java class that defines this bean object. Click on Browse... to browse for existing java class. <blank>
Parent Type the name of the parent bean definition. <blank>
Ignore errors When selected, the bean wizard disables the Finish button until all errors are resolved. To bypass error checking, deselect this option to proceed with bean creation with errors. This selection is remembered for subsequent bean wizard usages. This option appears in all pages of the bean wizard. Ignore errors disabled.

Properties and constructor args parameters page

You can skip this page by pressing Finish on the first page. Otherwise press Next to define property or constructor arg parameters.

Properties section

Action Description
Add Add a property node to the new bean.
Edit Edit the attributes of the selected property node.
Remove Remove the selected property node

Property node attributes:

Attribute Description
Name This attribute is required and it specifies the name of the property. This must correspond to a setter method in the specified bean class.
Value Specifies the value this property should be set to. Only one of value or ref can be specified.
Ref Specifies the name of the bean definition that holds the value of this property. Only one of value or ref can be specified.

Constructor args section

By default this section is collapsed. To expand this section, click on the header Constructor Args.

The combo box at the start of this section specifies which constructor of the specific bean class will be satisfied with the constructor-arg definitions. By default the combo box is empty. Once a selection is made, the corresponding number of constructor-arg will be added to the table, with the default <BLANK> value.

Action Description
Add Add a constructor-arg node to the new bean.
Edit Edit the attributes of the selected constructor-arg node.
Remove Remove the selected constructor-arg node

Constructor-arg node attributes:

Attribute Description
Value Specifies the value this property should be set to. Only one of value or ref can be specified.
Ref Specifies the name of the bean definition that holds the value of this property. Only one of value or ref can be specified.
